# Air Hockey Game  

A fast-paced and interactive **Air Hockey Game** built to deliver a fun browser-based gaming experience with smooth controls, real-time movement, and competitive gameplay.

##  Overview

This project is a simple yet engaging air hockey game where players compete to score goals by hitting the puck into the opponent’s side. It focuses on game mechanics, collision handling, score tracking, and responsive interaction.

##  Features

- Interactive air hockey gameplay
- Smooth paddle and puck movement
- Real-time collision detection
- Score tracking system
- Simple and clean user interface
- Fun and beginner-friendly project structure

##  Tech Stack

This project may use the following technologies:

- HTML
- CSS
- JavaScript

##  Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Hridesh-Thakur/Air-Hockey-Game.git
```

### 2. Open the project

Run the `index.html` file in your browser.

##  How to Play

- Start the game in your browser
- Control the paddle using the mouse or keyboard, depending on your implementation
- Hit the puck toward the opponent’s goal
- Defend your side and try to score maximum points
- The player with the higher score wins
